# mobile-layout-rem
> 这是一个使用 rem 为单位的移动端自适应布局工具。

## Installation

    npm install mobile-layout-rem

    // 或

    yarn add mobile-layout-rem

## Usage
```javascript
import initLayoutREM from 'mobile-layout-rem'

// 直接调用即可
initLayoutREM()
```

### 单位换算
本工具默认的是基于 750px 的设计稿的，因此换算公式为：

  20px = 20 / 100 = 0.2rem
